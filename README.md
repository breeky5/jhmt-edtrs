# jhmt-edtrs
This repo is for the JHMT RO BOH EDT Registration System Documentation

## Static Website

This repository contains a static website with the following pages:

- **Home** (`index.html`) - Landing page with system overview and key features
- **Terms of Service** (`terms-of-service.html`) - Legal terms and conditions
- **Privacy Policy** (`privacy-policy.html`) - Privacy and data protection information

### Running Locally

To view the website locally, you can use any web server. For example:

```bash
# Using Python 3
python3 -m http.server 8080

# Using Node.js
npx http-server -p 8080
```

Then open your browser to `http://localhost:8080`

### Features

- Responsive design that works on mobile and desktop
- Clean, professional styling
- Easy navigation between pages
- Comprehensive legal documentation
